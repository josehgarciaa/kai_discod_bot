# Local Chatbot

This project implements a local chatbot based on OpenAI models.

## Files and Structure
- **main.py**: Entry point of the application.
- **chat.py**: Core chat logic and ChatBot class.
- **storage.py**: Chat logs saving and loading functionality.
- **models.py**: Interface with OpenAI models.
- **admin.py**: Processes admin instructions.
- **utils.py**: Utility functions for the project.
- **config.py**: Configuration settings.
